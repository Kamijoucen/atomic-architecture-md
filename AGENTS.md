# AGENTS.md

本仓库为 Atomic Architecture（原子架构）规范的文档仓库，仅包含架构定义文档，用于共享和参考。

## 仓库说明

- **性质**: 纯文档仓库，无代码
- **核心文件**: `architecture.md`（架构规范定义）
- **辅助文件**: `README.md`（使用说明）
- **语言**: 中文

## 构建/测试命令

无需构建、lint 或测试。本仓库仅包含 Markdown 文档。


## 文档格式
- 内容使用中文编写
- 标题层级：# 主标题、## 章节、### 小节

## 文件说明

| 文件 | 用途 |
|------|------|
| `architecture.md` | 核心架构规范（禁止重命名） |
| `README.md` | AI 代理使用指南 |
| `AGENTS.md` | 本文件，代理操作参考 |

## 使用方式

其他项目引用本规范时：
1. AI 代理应阅读 `architecture.md` 理解规范
2. 将规范内容应用到目标项目代码中
3. 在目标项目创建/更新 `AGENTS.md` 时，内联写入规范要点

## Git 工作流

- 遵循常规 Git 提交规范
- 无特殊构建或发布流程
